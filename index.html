Seller Outreach System

Run seller outreach from one local command center.

Add sellers, generate personalized outreach, approve or suppress by channel, and send by text, email, or direct mail using Gmail, Google Voice workflow, and printable mail.

This build now includes

  • Persistent backend storage
  • Manual approval gate and auto-send option
  • Suppression controls per lead and channel
  • Gmail SMTP, Google Voice workflow, and mail export
Start the server with ruby server.rb, then open http://localhost:4567. Add a Gmail app password in .env when you are ready for live email.
